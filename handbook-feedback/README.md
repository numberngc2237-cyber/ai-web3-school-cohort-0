# Handbook Feedback

本目录用于记录学习过程中对 [AI × Web3 School Handbook](https://aiweb3.school/zh/handbook/) 的反馈和建议。

## 反馈内容类型

- 概念不清楚或需要更多解释
- 文档错误（错别字、链接失效、信息过时）
- 结构建议（章节顺序、分类方式）
- 缺少的案例、图解或代码示例
- 新增知识节点的建议

## 反馈模板

每条反馈单独一个文件，命名格式：`YYYY-MM-DD-简短描述.md`

```markdown
# 反馈：{{简短描述}}

- **日期**：YYYY-MM-DD
- **Handbook 页面**：https://aiweb3.school/zh/handbook/xxx.html
- **问题类型**：概念不清 / 文档错误 / 结构建议 / 缺少案例 / 其他

## 问题描述

<!-- 具体描述遇到的问题或发现的问题 -->

## 建议改法

<!-- 你建议如何改进 -->

## 来源

<!-- 这条反馈是在学习哪个章节时产生的 -->
```

## 反馈流程

1. 学习中发现问题 → 记录到本目录
2. 定期整理 → 提交到 [Handbook GitHub](https://github.com/lxdao-official/aiweb3-docs)
3. 跟踪反馈是否被采纳 → 更新反馈状态
